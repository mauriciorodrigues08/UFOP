O COMANDO FOR
- laço controlado por contador
- em um laço controlado por contador, os comandos (corpo do laço) são repetidos um número predeterminado de vezes.
- utilizado quando se sabe a quantidade de repetições a ser realizadas
- sintaxe:
  for(<const i>; <nº de repeticoes>; <incremento>) {
    <comandos>
  }

LAÇOS ANINHADOS
- é possível realizar laços dentro de laços, esses são os chamados laços aninhados
- ex: codigo que imprime uma cascata de valores

  for (int i = 0; i < 10; i++) {
    for (int j = 0; j < i; j++) {
      <comandos>
    }
  }
