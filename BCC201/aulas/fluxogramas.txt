FLUXOGRAMAS

1. O que é?
	- Um programa de computador que executa uma sequencia de instruções para realizar uma tarefa
	- Na programação estruturada, essas instruções podem ser classificadas em tres tipos:
		. Sequência
		. Decisão
		. Iteração

2. Representação
	- As instruções podem ser representadas textualmente ou graficamente por fluxogramas.
	- Facilita o entendimento e planejamento do código.

3. Estruturas
	- Compostos por uma sequencia enfileirada de instruções que são executadas uma após a outra.
	- As instruções são representadas em caixas
	- A ordem de realização das instruções é indicada por setas interligando as caixas.

4. Representações
	- Elipse: início e fim
	- Quadrado: instruções básicas
	- Losango: comandos de decisão/desvio
	- Seta voltando para a condição e saindo dependendo de um retorno: iteração/repetição/laço



PRIMEIRO PROGRAMA

1. Estrutura básica de um código em C
	<inclusão de bibliotecas>
	
	int main (<declaração de parâmetros>){
		comando_1;
		comando_2;
		...
		comando_n;
		return 0;
	}

2. Comentários no Código
	- Para utilizar comentário de uma linha: //
	- Para utilizar comentário de 2+ linhas: /**/ (abre e fecha comentário)
	- Obrigatório nos códigos da aula: comentário com nome e matrícula

3. Saída do Código
	- É o que aparecerá na tela após a execução do código.

4. Como incluir bibliotecas em C
	- Utilizamos o comando "include"
	#include <nome da biblioteca>

5. Extensão do Programa em C
	- Para salvar o código utilizamos a extensão ".c"
	- Ex.: codigo001.c
